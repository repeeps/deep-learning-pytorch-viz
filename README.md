# 딥러닝, 눈으로 보기 (Deep Learning, Visualized)

텐서부터 LLM까지 — 딥러닝의 핵심 개념을 **브라우저에서 실제로 계산되는 숫자**로 직접 돌려보며 익히는 인터랙티브 학습 자료입니다. 가짜 도식이 아니라 모든 행렬곱·softmax·경사하강이 실시간으로 동작합니다.

> 직접 만든 **비영리 학습용 자료**입니다. 특정 교재·강의와 무관하며, 설명·예시·코드는 모두 독자적으로 제작했습니다.

## 시작하기

빌드·의존성 없음. `index.html` 을 브라우저로 열면 끝입니다.

```bash

# 또는 간단한 로컬 서버
```

## 설계 원칙

- **실제 계산**: 결과값 하드코딩 없이 JS가 매번 계산 (입력 샘플 데이터만 고정)
- **단계별 분해**: 각 챕터 4스텝, 한 줄 직관 → 인터랙션 → 결론
- **일관된 시각 언어**: 색 코드(입력/Q=시안, K=코랄, V=바이올렛, 결과=그린)와 행렬 렌더러 공유
